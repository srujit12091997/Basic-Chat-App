## ⚙️ Installation

1. Clone the repository:
```bash
git clone https://github.com/srujit12091997/modern-chat-app.git
```

2. Navigate to project directory:
```bash
cd modern-chat-app
```

3. Open index.html in your preferred browser:
```bash
# For Linux/Mac
open index.html

# For Windows
start index.html
```

## 📖 Usage

1. Open the application in your browser
2. Start typing your message in the input field
3. Press Enter or click the Send button to send messages
4. Watch for automated responses
5. Enjoy the smooth chat experience!
